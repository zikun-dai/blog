快速指令

未修改插件或主题时无需 clean。 npm install 新插件，或修改了 _config.yml，建议先 clean 一次。

# 本地预览
hexo clean && hexo g && hexo s
# 直接部署
hexo clean && hexo g && hexo d
# 日常新增文档时，可跳过 clean
hexo g -d

单步指令

新建文章

hexo new "My New Post"

More info: Writing

清理缓存

启动本地服务

hexo server

More info: Server

生成静态文件

# equal

部署到远端
