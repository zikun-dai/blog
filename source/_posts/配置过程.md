### 修改完成后的效果

修改完保存后，你可以再次运行 `hexo clean && hexo g && hexo server`。

如果配置正确，打开一篇文章的详情页，拉到最底部，你应该能看到一个评论框（或者显示 "Login to comment" 的按钮）。

> **注意**：
>
> 1. 请确保你在 GitHub 上已经创建了名为 `blog` 的仓库（或者你填写的任何仓库名）。
> 2. 该仓库必须是 **Public（公开）** 的，否则访客无法查看和评论。
> 3. 评论插件通常在 `localhost` 本地预览时也能加载，但你需要登录 GitHub 授权一下 Utterances app。

```
hexo clean && hexo g && hexo s
```

```
hexo clean && hexo g && hexo d
```



```bash
hexo clean   # 清除缓存
hexo g       # 生成静态页面 (Generate)
hexo d       # 部署到 GitHub (Deploy)
```
